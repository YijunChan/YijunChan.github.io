参加腾讯Web前端大会是一种什么什么样的体验？

发表于 2017-07-15

前言

迟来的体验总结；
不多说，先上图。
(最近家里的电脑又重装系统了，开发环境神马的最烦了~)









总结和收获

  1. 本着增长见识及扩展人脉的念头去参会了，认识了一大群来自羊城的同行；
  2. 技术方面，第一次参加这种技术型会议，以为会干货慢慢，结果各位大牛只是粗略讲解一下一些前沿技术的应用，demo都少，感觉相比16年广州的css开发者大会略差
  3. 在职业规划方面，这方面倒是收获不少，听取大牛们分享他们的一些成长经历，对自己人生规划倒是有促进作用。

感谢

特别感谢我司赞助门票，本来确实是有意参加的，但是门票确实高昂（原价998），最后公司领导主动提出让我去参会，非常感谢。
